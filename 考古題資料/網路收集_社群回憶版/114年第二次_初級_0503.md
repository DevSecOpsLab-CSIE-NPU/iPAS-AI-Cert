## 科目二：生成式 AI 應用與規劃（約 23 題）

1. 資料飄移（Data Drift）：輸入特徵統計分布隨時間變動
2. 提示洩漏（Prompt Leakage）：模型暴露不應看到的資訊
3. 提示注入攻擊（Prompt Injection）：操控攻擊導致偏離任務
4. 「非」提示工程：AutoML 是整個 ML 流程自動化
5. 文本生成：提示優化（2 題）
6. 「非」程式碼生成 AI：Sora 用於文字轉影片
7. 生成音樂 AI：Suno AI
8. TAIDE：台灣智慧對話引擎
9. No code / Low code：約 5 題
10. No code 應用與限制：3 題
11. 生成式 AI 風險：5 題
12. GPT-4o 敘述：部分資料來源由合作夥伴
13. 回歸模型：溫度預測飲料銷量
14. 線性回歸應用：1 題
15. 圖片解析度計算：2048×2048 @ 300DPI ≈ 6.83 英寸
16. 補習班評語平台：使用 ChatGPT/OpenAI API + 表單
17. NotebookLM：Google 工具，生成摘要與播客
18. CLIP 多模態模型：圖片與文字映射到同一嵌入空間
19. 對抗性實驗：測試模型在極端條件下表現
20. 延遲性實驗：測試不同元件的時間消耗
21. AI Agent 核心功能：「不」包含發明新技術
22. AI 環境適應能力：泛化能力
23. RAG vs 微調：RAG 時間快（無需重新訓練）
