----------------------------------------------------------------
1. 資料類型與儲存
----------------------------------------------------------------
* 結構化資料: 表格 (SQL DB)
* 半結構化: JSON, XML, CSV (彈性 schema)
* 非結構化: 影像、影片、語音、自由文本
* CSV: 純文字、逗號分隔、無型別資訊
* 資料倉儲 Data Warehouse: 結構化、已清理、適合分析
* 資料湖 Data Lake: 多類型原始資料、彈性高
* 資料湖屋 Data Lakehouse: 兩者混合 (Databricks)

【陷阱題】影像是非結構化, 不是半結構化

----------------------------------------------------------------
2. 資料清理
----------------------------------------------------------------
* 缺失值: 刪除 / 平均/中位/眾數 / 預測補值
* 異常值偵測:
    - Z-score: |Z| > 3 視為異常
    - IQR 法: < Q1-1.5*IQR 或 > Q3+1.5*IQR
    - 箱型圖視覺化
* 異常值處理:
    - 直接刪除 (要謹慎)
    - Log 轉換壓縮極端值
    - Winsorize 截斷
    - ★ 標準差/平均數本身不是處理方法, 是檢測指標
* 資料飄移 Data Drift: 輸入特徵分布隨時間變化 → 模型需重訓

----------------------------------------------------------------
3. 敘述統計
----------------------------------------------------------------
* 集中趨勢 Central Tendency:
    - 平均數 Mean: 受極端值影響
    - 中位數 Median: 抗極端值
    - 眾數 Mode: 出現最多次
* 變異性 Variability:
    - 標準差 Std / 變異數 Variance
    - 全距 Range
    - 四分位距 IQR
* 中位數是「集中趨勢」, 不是「變異性」(★陷阱)
* 偏態 Skewness: 右偏 → 平均 > 中位 (受極端高值拉高)
* 常態分布: 對稱鐘形, 約 68%-95%-99.7% 規則

----------------------------------------------------------------
4. 視覺化選擇 (★情境配對題)
----------------------------------------------------------------
* 散佈圖 Scatter: 兩數值變數的關係 (身高 vs 體重)
* 直方圖 Histogram: 單一連續變數的分布、偏態、多峰
* 箱型圖 Boxplot: 分布摘要、找異常值、跨組比較
* 折線圖 Line: 時間序列趨勢
* 長條圖 Bar: 類別比較
* 圓餅圖: 占比 (不適合多類別)
* 熱圖 Heatmap: 兩變數關聯強度、相關矩陣
* 雷達圖: 多維度比較 (★ 不適合判斷集中趨勢)

【口訣】
  分布看直方/箱型 → 關係看散佈/熱圖 → 時間看折線 → 類別看長條

----------------------------------------------------------------
5. 大數據工具與生命週期
----------------------------------------------------------------
* MapReduce: Map 切分 → Reduce 彙總 (Hadoop 核心)
* Hadoop: HDFS 儲存 + MapReduce 運算
* Spark: 記憶體運算, 比 Hadoop MapReduce 快
* ETL: Extract → Transform → Load
* ELT: 順序對調, 適合資料湖
* MLOps 生命週期:
    1. 資料收集 → 2. 模型訓練 → 3. 部署
    → 4. 監控 → 5. 重新訓練 (循環)
* MLflow: 管理 ML 實驗、模型版本、生命週期
* HITL (Human-in-the-Loop): 人類專家參與訓練/校正
