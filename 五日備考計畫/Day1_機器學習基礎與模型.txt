----------------------------------------------------------------
1. 學習典範分類 (常考: 每場 3~5 題)
----------------------------------------------------------------
* 監督式 Supervised: 有標籤 → 分類 / 迴歸
    例: 預測客戶流失、信用評分、垃圾郵件分類
* 非監督式 Unsupervised: 無標籤 → 分群 / 降維
    例: 顧客分群、異常檢測、PCA 降維
* 半監督式 Semi-supervised: 少量標籤 + 大量未標籤
    例: 醫療影像僅部分有專家標註
* 強化學習 Reinforcement: 獎勵/懲罰訊號
    例: 自駕車、AlphaGo、推薦系統最佳化

【陷阱題】"在自駕車模擬中根據是否安全通過獲得獎勵" → 強化學習 (非非監督)

----------------------------------------------------------------
2. 經典演算法
----------------------------------------------------------------
* KNN: 基於距離、惰性學習、K 值決定鄰居數
* 決策樹: 可解釋、易過擬合、需剪枝
* 隨機森林: Bagging 集成、降低變異、抗過擬合
* SVM: 找最大邊界、核技巧處理非線性 (注意: 非深度學習)
* 線性迴歸 / 邏輯迴歸: 連續值 / 二分類
* K-means: 距離 / DBSCAN: 密度 (兩者差異必考)
* 樸素貝葉斯: 機率、文本分類常用

----------------------------------------------------------------
3. 模型評估指標 (公式必背)
----------------------------------------------------------------
* 混淆矩陣: TP, FP, TN, FN
* Accuracy = (TP+TN) / 全部
* Precision (精確率) = TP / (TP+FP)  ← 寧缺勿濫
* Recall (召回率) = TP / (TP+FN)     ← 寧錯勿漏
* F1 = 2 * P * R / (P+R)             ← 調和平均
* ROC / AUC: 衡量二分類門檻不變下的整體能力
* 迴歸: MAE, MSE, RMSE, R²

【情境題】醫療癌症篩檢: 重 Recall (寧誤判也不漏)
         垃圾郵件: 重 Precision (寧漏也別誤刪正常信)

----------------------------------------------------------------
4. 過擬合 vs 欠擬合
----------------------------------------------------------------
* 偏差 Bias: 模型太簡單 → 欠擬合
* 變異 Variance: 模型太複雜 → 過擬合
* 對策: 交叉驗證 / 正則化 (L1, L2) / Dropout / 早停 /
        增加資料 / 減少特徵 / 換較簡單模型
* K-fold 交叉驗證: 切 K 份輪流當驗證集

----------------------------------------------------------------
5. 特徵工程與選擇
----------------------------------------------------------------
* 特徵縮放: 標準化 (Z-score) / 正規化 (Min-Max)
* 類別編碼: One-hot (無序) / Label Encoding (有序)
* 缺失值: 刪除 / 平均/中位/眾數補值 / 模型預測
* 特徵選擇:
    - 過濾法 (Filter): 統計量篩選
    - 包裝法 (Wrapper): 模型評估
    - 嵌入法 (Embedded): L1 自動選擇
* 監督式特徵選擇: 看與目標變數的相關性
* PCA 降維: 找最大變異方向
